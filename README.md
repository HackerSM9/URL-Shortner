# URL-Shortner
URL-Shortner in Termux using TinyURL.com
### Usage
1) **Short URL/LINK** [ To Shorten the URL/Link ]
2) **More tools** [ More tools for Termux ]
3) **About** [ About Author ]
4) **Upgrade** [ for Update ]
