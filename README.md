# URL-Shortner
URL-Shortner in Termux using python3.
## Installation
```
cd $HOME 
pkg install git
pkg install python3
git clone https://github.com/HackerSM9/URL-Shortner
cd URL-Shortner
python3 setup.py
```
### Usage
1) **Short URL/LINK** [ To Shorten the URL/Link ]
2) **More tools** [ More tools for Termux ]
3) **About** [ About Author ]
4) **Upgrade** [ for Update ]
